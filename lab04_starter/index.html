The Squirrley Square Game

Click the "Add Square" button to add a black square to the screen

The square has an id number equal to the number of squares on the page at the time you click the button

Click the squares to change thier colors and make patterns

Removing squares is a little more difficult. If you double click a square with an even id number, the square after it will be removed. If you double click a square with an odd id number, the square before it will be removed.
